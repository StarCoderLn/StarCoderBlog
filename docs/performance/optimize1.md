2. 为什么要做性能优化？

- 57%的用户更在乎网⻚在3秒内是否完成加载。

- 52%的在线用户认为网⻚打开速度影响到他们对网站的忠实度。

- 每慢1秒造成⻚面 PV 降低11%，用户满意度也随之降低降低16%。

- 近半数移动用户因为在10秒内仍未打开⻚面从而放弃。
